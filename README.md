A desktop app built with pygame in python for the ui and using a free tier google gemini api for the ai. It uses a custom design language in which the gemini llm responses and which is converted into shapes on the canvas. There is an api key hardcoded in the python file, please use this one only for quick testing and not real use, you can very easily make your own at https://aistudio.google.com and paste it at the top of the python file.
